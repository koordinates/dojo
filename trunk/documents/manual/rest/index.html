General purpose libraries

Dojo includes some core libraries to make it easier to work with JavaScript, HTML, the DOM:
  • dojo: The Dojo bootstrap.
  • dojo.lang: Utility routines to make JavaScript easier to use.
  • dojo.string: String manipulation routines.
  • dojo.dom: DOM manipulation routines.
  • dojo.style: CSS Style manipulation routines.
  • dojo.html: HTML specific operations
  • dojo.date: Date manipulation. (undocumented)
  • dojo.logging.Logger: Logging library (undocumented).
  • dojo.profile: JS Profiler (undocumented)
  • dojo.reflect: Reflection API (undocumented)
  • dojo.regexp: Regular Expression generators (undocumented)
  • dojo.undo.Manager: Undo management (undocumented)
  • dojo.event: The Aspect Oriented Programming-inspired event system (undocumented, but see this introduction)

Data Structures

The dojo.collections package has a few sub-parts:
  • Dictionary: a dictionary/hash object
  • ArrayList: an extended version of the core ECMAScript Array object.
  • Queue: a basic implementation of a typical queue (FIFO) object.
  • SortedList: An indexed/ordered Dictionary
  • Set: an object that provides basic set operations.
  • Stack: a basic implementation of a standard Stack (FILO).

Web I/O

  • dojo.io: Platform independent input/output APIs (also see this)
  • dojo.rpc: RPC, the Dojo way
  • dojo.json: JSON, the Dojo way

Visual Web

  • dojo.animation.Animation: Animation supprt
  • dojo.dnd: Drag and Drop support
  • dojo.validate: Data validation methods (isText, isNumber, isValidDate, etc..)
  • dojo.fx: Fading, exploding visual effects (undocumented)
  • dojo.graphics.Colorspace: Colorspace manipulation (undocumented)
  • dojo.graphics.color: color manipulation (undocumented)
  • dojo.svg: SVG library (undocumented)

Math and Crypto

  • dojo.crypto: Cryptographic routines.
  • dojo.math.Math: Math library (undocumented)
  • dojo.math.curves: Curve generation library (undocumented)
  • dojo.math.matrix: Linear Algebra (undocumented)
  • dojo.math.points: Point manipulations (undocumented)

Other

  • dojo.storage: The storage system implements a local durable cache.
  • dojo.xml.Parse: a first pass XML to JS parser.
  • dojo.uri: URI/URL manipulation routines
