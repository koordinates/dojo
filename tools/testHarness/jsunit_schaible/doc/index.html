JsUnit

1.2.beta.1

Description

JsUnit is a simple framework to write repeatable tests in JavaScript. It is an instance of the xUnit architecture for unit testing frameworks. JsUnit is a port of JUnit 3.8.1 originally written by Erich Gamma and Kent Beck. It covers the core system and the examples.

For further information follow the links at Portland Pattern Repository for Testing Frameworks and the homepage of JUnit.

JsUnit supports a number of applications. See Compatibility for a reference and the different language versions.

JsUnit uses a Perl script to support JavaDoc comments for JavaScript. This script js2doxy.pl is used by some people on its own. It is linked also at the Doxygen homepage. See Internals for further description. Get the latest version directly form the CVS repository of JsUnit.
