The Dojo Toolkit

Dojo is an Open Source effort to create a UI toolkit that allows a larger number of web application authors to easily use the rich capabilities of modern browsers.

Members of the team building Dojo have helped to build nWidgets, Burst, and f(m). To find out how you can be part of that team, check out the project wiki or see the project status and roadmapto find out where we're going.
